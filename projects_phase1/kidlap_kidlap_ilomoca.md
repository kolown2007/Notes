Kidlap-Kidlap is a sound reactive application. 

technical details: 
The application is very simple
1. install the app
2. plug the cpu to a monitor or projector
3. connect the microphone
4. project the visuals depends on the preference of the exhibit designer/ curator

**requirements:**

- a stable CPU (any OS) - to run the application
- computer microphone - usb mic to connect to cpu to gather sound
- projector - to visualize the application
- stable internet - for app updates and monitoring
